05 · UNDER THE HOOD

What's wired up.

Every system is a real class in this single-file prototype. Inspect the source — no build step, no framework.

CLASS

APIService

PokéAPI fetcher with in-memory + localStorage cache. Sprite priority chain Showdown → Gen V → Front → Home → Official.

CLASS

AudioManager

Procedural chiptune music + Web Audio SFX (spawn, jump, capture, escape). Plays cries.latest when present.

CLASS

ARScene

Three.js + WebXR hit-test, animated reticle, controller-tap placement, virtual props on the detected plane.

CLASS

FallbackScene

Orbitable 3D arena with low-poly props, click-to-place, holographic floor, and the same physics + capture loop.

CLASS

CreatureSprite

THREE.Sprite billboard with type aura, scale from real height, idle bobbing, jump, hit-flash, capture shake.

CLASS

SimplePhysics

Gravity, ground collision, prop colliders, arena bounds, bounce, ground-tracking fake shadow.

CLASS

CaptureSystem

Chance from base HP, weight, simulated rarity, reticle distance + type. Persists inventory to localStorage.

EFFECTS

Type FX

Eleven type auras — electric sparks, fire glow, water bubbles, grass leaves, ghost smoke, psychic rings…

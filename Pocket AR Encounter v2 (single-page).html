▸ 04 · UNDER THE HOOD

Systems.

Every system below is a real JavaScript class in this single file — no build step.

CLASS

APIService

PokéAPI fetcher with in-memory + localStorage cache. Sprite chain: Showdown → Gen V → Front → Home → Official.

CLASS

AudioManager

Procedural pad + arpeggio music, Web Audio SFX (scan, spawn, ball-throw, shake, capture, escape), and cries.latest playback.

CLASS

ARScene

Three.js + WebXR hit-test, animated reticle, controller-tap placement, virtual props anchored to the scanned plane.

CLASS

FallbackScene

Pixel-art battle stage with orbit camera, click-to-place, virtual grass/rocks/portal, same physics and capture flow.

CLASS

CreatureSprite

Billboard sprite with type aura, scale from real height, idle bobbing, jump, hit-flash, capture shake, flee dash.

CLASS

SimplePhysics

Gravity, ground collision, prop colliders, arena bounds, bounce, ground-tracking fake shadow.

CLASS

CaptureSystem

Chance based on base HP, weight, simulated rarity, reticle distance + type. Persists inventory to localStorage.

FX

Type Auras

Eleven elemental halos — electric sparks, fire glow, water bubbles, grass leaves, ghost mist, psychic rings, dragon spiral…

EDUCATIONAL PROTOTYPE · NOT OFFICIAL

This is a non-commercial fan / research prototype. Pokémon and all related assets belong to their respective rights-holders.

  • No official logos, music, or audio extracted from games.
  • Sprite + species data from the public PokéAPI for prototyping only.
  • For public / commercial release, swap creatures, sprites and audio for original licensed assets.
  • Music is generated in-browser via the Web Audio API. SFX are procedural.
  • Responses are cached to respect API limits.
